Features used on this site.


Feature What it does Where it is used
Columns Columns help divide the page into neat and user friendly rows Columns can be seen used throughout most pages in this CA
Carousels Carousels are image gallerys that show at least one picture at a time and automatically or manually scroll to another image Can be seen on the Grades page and the About page
Jumbotron Holds information in a neat grey box Can be found on various pages
Buttons More user friendly decorated links Can be found on various pages
Modals Click a button/image/link and a popup box appears with information inside of it Can be found on the Projects page
Dropdowns Click a button and a dropdown menu appears Can be found on the Navbar
Show More/Less Click a button and collapsable information appears Can be found on the Projects page
Tables Divides information into neat tables Can be found on the Grades page and is also used on this page
Embedded Google Maps Location Small window that displays the location of I.T. Tallaght Can be found on the Contact page
Href MailTo: When the email is clicked an email popup appears on mobiles with the subject and target email address already entered. Can be found on the Contact page
Href Tel: When a number is clicked a popup appears on mobiles asking if the user would like to ring the number Can be found on the Contact page
Jump To: Anchor Click a link or button and jump to a certain section on the page Can be found on the Experience page
Nav Tabs/Pills I used the nav tabs to let the user select a certain year 1, 2, 3, or 4 Can be found on the Grades page
Image Transitions The transition I used was transitioning one image to another This transition can be seen by hovering over the images on the Index page
Image Animations The animation I used was making images expand slightly This can be found by hovering on the project images on the Projects page
Font Awesome Icons Specialised icons for different subject types Can be found on the Contact page
Tooltips Hover over a piece of information and more information appears Can be found by hovering over Module title on the tables on the Grades page
